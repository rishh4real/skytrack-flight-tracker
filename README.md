# ✈️ SkyTrack – Real Time Flight Tracker

## 📌 Project Overview

SkyTrack is a modern web application that allows users to view real-time flight information from around the world.
The application fetches live aviation data using a public API and displays it in a clean and interactive dashboard interface.

This project demonstrates the use of JavaScript API integration, dynamic UI rendering, and user-friendly design principles.

---

## 🎯 Purpose of the Project

The goal of this project is to build a responsive web application that helps users:

* Track live flight status
* Search flights by flight number
* Filter and sort flight information
* View airline and route details in an organized way

The project also helps in understanding real-world frontend development concepts such as asynchronous data fetching, DOM manipulation, and UI design.

---

## 🌐 Public API Used

This application uses the **AviationStack API** to fetch real-time flight data.

API Features Used:

* Live flight status data
* Airline and airport information
* Departure and arrival details
